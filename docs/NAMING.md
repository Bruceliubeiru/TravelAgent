# Naming

Trip GEO Growth OS is the system name.

TravelAgent is the Mini Program App inside Trip GEO Growth OS. It is not the system name.

H5 Landing App is the independent indexable growth page layer.

Ops Center is an optional operations plugin.

Device Gateway is an optional device execution abstraction.

Moyunteng is one Device Adapter only. It is not the system name and not a core dependency.
