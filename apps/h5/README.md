# H5 Landing App

H5 Landing App 是 Trip GEO Growth OS 的独立可收录增长页面层。

## 定位

- 面向 Google SEO 与 AI Search / GEO 收录
- 承载 POI Landing Page、FAQ Page、Compare Page
- 最终跳转 Trip 官方预订页
- 负责 UTM / trip_sub 等归因参数透传

## 边界

- 不依赖小程序页面逻辑
- 不依赖 Ops Center
- 不依赖 Device Gateway
- 不直接接入任何设备供应商
- 不实现真实支付或履约

## 当前状态

本目录当前仅为空壳，用于建立独立 H5 应用边界。完整页面、SEO 内容生成、Schema、部署配置将在后续迭代中补充。
