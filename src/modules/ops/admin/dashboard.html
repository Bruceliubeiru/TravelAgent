API Contract

This static page is intentionally not wired into the existing mini program.

  • POST /ops/tasks create task
  • GET /ops/tasks/:taskId get task
  • POST /ops/tasks/:taskId/run run pending task
  • POST /ops/tasks/:taskId/callback receive callback
  • GET /ops/dashboard dashboard summary
